This project uses Windows-1251 encoding.
Yon need to extract original files from ZIP file to save original file encoding.
And then try IDEA > File > Settings > Editor > Console > Default Encoding <System Default: windows-1251>
Also IDEA > File > Settings > Editor > File Encodings > Global <windows-1251> and Project Encodings <System Default: windows-1251>

Good Luck! And Let's Play!
